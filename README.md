# 💳 Razorpay Clone

A fully responsive **Razorpay landing page clone** built using **HTML** and **Tailwind CSS**.  
This project replicates the UI/UX of the Razorpay website with modern design practices and responsive layout.

🔗 **Live Demo:** https://razorpay-myclone.netlify.app/

---

## 🚀 Features

- ⚡ Fully responsive design (Mobile + Tablet + Desktop)
- 🎨 Pixel-perfect UI inspired by Razorpay
- 💨 Built with Tailwind CSS utility-first framework
- 📱 Modern layout and clean design
- 🌐 Fast and optimized performance

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS**
- **CSS3**
